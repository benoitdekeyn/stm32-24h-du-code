# Tutoriels (STM32MP1)

Ce dossier contient des tutoriels courts, orientés « pas-à-pas », pour gagner du temps sur les manipulations récurrentes (WSL, SSH, carte SD, flash de l’OS, compilation C).

## Conventions utilisées

- Les commandes à copier/coller sont dans des blocs `bash` (WSL/Linux) ou `powershell` (Windows).
- Les chemins et noms d’interface réseau sont des exemples : adapte-les à ton PC (ex: `Ethernet 4`).
